🧠 train.csv — Dados de Treinamento (Histórico)

Este arquivo contém dados de mercado históricos. A cobertura é muito extensa (vários anos ou décadas), mas nas datas mais antigas há valores ausentes, já que nem todos os indicadores existiam ou eram medidos naquela época.

📊 Colunas do train.csv: Coluna Descrição - date_id Identificador único de cada dia de negociação. É um número inteiro que representa uma data específica. - M* Market Dynamics / Technical features — características técnicas de mercado, como médias móveis, volumes, indicadores técnicos (RSI, MACD, etc). - E* Macro Economic features — variáveis macroeconômicas, como PIB, inflação, desemprego, índices de confiança. - I* Interest Rate features — taxas de juros e instrumentos de renda fixa, como “federal funds rate”, “treasury yield”, spreads, etc. - P* Price / Valuation features — métricas de preço e avaliação, como P/E (preço/lucro), P/B (preço/valor patrimonial), etc. - V* Volatility features — indicadores de volatilidade, como VIX ou medidas estatísticas de variação de preço. - S* Sentiment features — medidas de sentimento do mercado, extraídas de notícias, redes sociais ou relatórios. - MOM* Momentum features — indicadores de tendência, como retornos acumulados, força relativa, etc. - D* Dummy/Binary features — variáveis binárias (0/1) que podem representar eventos (ex: feriado, anúncio do FED, etc). - forward_returns Retorno futuro diário — o retorno obtido ao comprar o S&P 500 e vendê-lo no dia seguinte. É a variável-alvo que queremos prever. - risk_free_rate Taxa livre de risco — normalmente a taxa de fundos federais (Fed Funds Rate), usada como base para cálculos de excesso de retorno. - market_forward_excess_returns Retorno futuro em excesso — diferença entre o retorno futuro real e o retorno esperado (média móvel dos últimos 5 anos). Este valor é “winsorizado” (valores extremos ajustados) com base na MAD (Desvio Absoluto da Mediana) com um limite de 4 desvios.

def plot_rolling_means_grouped(df, groupby_cols, value_cols, agg='mean',
                               window=30, min_periods=1, max_cols=4,
                               figsize_per_ax=(5,3), title_fmt="{col} (rolling={w})",
                               sharex=False, sharey=False, style_kwargs=None,
                               numeric_only=True):
    """
    Agrupa o DataFrame por `groupby_cols`, agrega as `value_cols` (por ex. mean),
    e plota médias móveis das colunas agregadas em um grid.

    Parâmetros:
    - df: pandas DataFrame contendo as colunas de agrupamento e as colunas de valor.
    - groupby_cols: coluna única ou lista de colunas para agrupar (ex: ['date_id']).
    - value_cols: lista de colunas a agregar e plotar (ex: ['forward_returns','other_col']).
    - agg: função de agregação suportada por pandas (string ou callable), default 'mean'.
    - window: janela rolling (número de amostras).
    - min_periods: min_periods para rolling.
    - max_cols: número máximo de subplots por linha.
    - figsize_per_ax: (largura, altura) por subplot em polegadas.
    - title_fmt: formato do título (usa {col} e {w}).
    - sharex, sharey: compartilhar eixos.
    - style_kwargs: kwargs para a linha da média móvel.
    - numeric_only: se True, filtra somente colunas numéricas de value_cols.

    Retorna:
    - fig, axes, meta onde meta é lista de dicionários com info de cada coluna agregada.
    """
    # Validacões básicas
    if isinstance(groupby_cols, (str, int)):
        groupby_cols = [groupby_cols]
    if isinstance(value_cols, (str, int)):
        value_cols = [value_cols]

    # Agregar
    grouped = df.groupby(groupby_cols)[value_cols].agg(agg)

    # Se o resultado tem índice composto, garantir que index seja temporal/ordenado
    grouped = grouped.sort_index()

    # Se for MultiIndex com mais de um nível e queremos plotar por nível, ajustar conforme necessário.
    # Aqui assumimos que index é `date_id` ou que seu primeiro nível é temporal sequencial.

    # Transformar para DataFrame simples (caso agg retorne Series quando value_cols length == 1)
    if isinstance(grouped, pd.Series):
        grouped = grouped.to_frame(name=value_cols[0])

    # Filtrar colunas numéricas se solicitado
    if numeric_only:
        cols = grouped.select_dtypes(include=[np.number]).columns.tolist()
    else:
        cols = grouped.columns.tolist()

    if len(cols) == 0:
        raise ValueError("Nenhuma coluna numérica encontrada após agregação para plotar")

    # Preparar grid
    n_cols = min(max_cols, len(cols))
    n_rows = math.ceil(len(cols) / n_cols)
    fig, axes = plt.subplots(n_rows, n_cols,
                             figsize=(figsize_per_ax[0]*n_cols, figsize_per_ax[1]*n_rows),
                             squeeze=False, sharex=sharex, sharey=sharey)
    axes_flat = axes.flatten()

    if style_kwargs is None:
        style_kwargs = {'color': 'tab:green', 'linewidth': 1.5}

    meta = []
    for i, col in enumerate(cols):
        ax = axes_flat[i]
        series = grouped[col]

        rolling = series.rolling(window=window, min_periods=min_periods).mean()

        ax.plot(series.index, series.values, alpha=0.15, color='gray', linewidth=0.7)
        ax.plot(rolling.index, rolling.values, label=f'rolling({window})', **style_kwargs)

        ax.set_title(title_fmt.format(col=col, w=window), fontsize=10)
        ax.grid(True)
        ax.legend(frameon=False, fontsize=8)

        non_null = series.dropna()
        meta.append({
            'col': col,
            'n_non_null': int(non_null.shape[0]),
            'first_index': non_null.index.min() if non_null.shape[0] else None,
            'last_index': non_null.index.max() if non_null.shape[0] else None,
            'window': int(window)
        })

    # Remover eixos vazios
    for j in range(len(cols), len(axes_flat)):
        fig.delaxes(axes_flat[j])

    plt.tight_layout()
    plt.show()

    return fig, axes, meta

##Processos Estacionários

Teorema 3.1 (Decomposição de Wold)

Considere um processo estacionário qualquer, \(y_t\). Tal processo pode ser representado por dois processos mutuamente não correlacionados - um puramente determinístico, outro puramente estocástico - e ser escrito como um MA(\(\infty\)):

\(Y_t = U_t + d_t\)

em que:

  1. \(U_t\) e \(d_t\) não são correlacionados;
  2. \(U_t\) é regular, sendo representado por: \(U_t = \sum_{s=0}^{\infty} \psi_s \epsilon_{t-s}\) \(\psi_0 = 1\); \(\sum_{s=0}^{\infty} \psi_s^2 < \infty\); \(\epsilon_t \sim RB(0, \sigma^2)\), sendo \(E(\epsilon_t d_s) = 0, \forall s,t\). Além disso, a sequência \(\{\psi_s\}_{s=0}^\infty\) e o processo \(\{\epsilon_t\}\) são unicamente determinados.
  3. \(d_t\) é singular no sentido de que pode ser previsto a partir do próprio passado com variância de predição nula.

Embora a decomposição seja linear mesmo para processos não lineares, o que significa que se pode aproximar um processo não linear por um linear, é claro que o processo não linear tende a aproximar melhor o processo estacionário.

Ruído Branco (White Noise)

\(y_t = \epsilon_t\)

Onde \(\epsilon_t\) é um ruído branco com média zero e variância constante \(\sigma^2\).

Processo Autorregressivo (AR(p))

\(y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \dots + \phi_p y_{t-p} + \epsilon_t\)

Onde \(c\) é uma constante, \(\phi_i\) são os coeficientes autorregressivos, e \(\epsilon_t\) é um ruído branco.

Processo de Média Móvel (MA(q))

\(y_t = \mu + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \dots + \theta_q \epsilon_{t-q} + \epsilon_t\)

Onde \(\mu\) é a média da série, \(\theta_i\) são os coeficientes de média móvel, e \(\epsilon_t\) é um ruído branco.

Processo Autorregressivo de Média Móvel (ARMA(p, q))

\(y_t = c + \phi_1 y_{t-1} + \dots + \phi_p y_{t-p} + \theta_1 \epsilon_{t-1} + \dots + \theta_q \epsilon_{t-q} + \epsilon_t\)

Onde \(c\) é uma constante, \(\phi_i\) são os coeficientes AR, \(\theta_i\) são os coeficientes MA, e \(\epsilon_t\) é um ruído branco.

Aqui estão as fórmulas para os modelos de séries temporais:

Ruído Branco (White Noise)

\(y_t = \epsilon_t\)

Onde \(\epsilon_t\) é um ruído branco com média zero e variância constante \(\sigma^2\).

Processo Autorregressivo (AR(p))

\(y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \dots + \phi_p y_{t-p} + \epsilon_t\)

Onde \(c\) é uma constante, \(\phi_i\) são os coeficientes autorregressivos, e \(\epsilon_t\) é um ruído branco.

Processo de Média Móvel (MA(q))

\(y_t = \mu + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \dots + \theta_q \epsilon_{t-q} + \epsilon_t\)

Onde \(\mu\) é a média da série, \(\theta_i\) são os coeficientes de média móvel, e \(\epsilon_t\) é um ruído branco.

Processo Autorregressivo de Média Móvel (ARMA(p, q))

\(y_t = c + \phi_1 y_{t-1} + \dots + \phi_p y_{t-p} + \theta_1 \epsilon_{t-1} + \dots + \theta_q \epsilon_{t-q} + \epsilon_t\)

Onde \(c\) é uma constante, \(\phi_i\) são os coeficientes AR, \(\theta_i\) são os coeficientes MA, e \(\epsilon_t\) é um ruído branco.

Processo Autorregressivo Integrado de Média Móvel (ARIMA(p, d, q))

Um processo ARIMA(p, d, q) é um processo ARMA(p, q) aplicado à série diferenciada \(d\) vezes. Seja \(y_t\) a série original, e \(\Delta^d y_t\) a série diferenciada \(d\) vezes.

\(\Delta^d y_t = c + \phi_1 \Delta^d y_{t-1} + \dots + \phi_p \Delta^d y_{t-p} + \theta_1 \epsilon_{t-1} + \dots + \theta_q \epsilon_{t-q} + \epsilon_t\)

Onde \(\Delta\) é o operador de diferença, \(\Delta y_t = y_t - y_{t-1}\).

Processo Autorregressivo Sazonal Integrado de Média Móvel (SARIMA(p, d, q)(P, D, Q)s)

Um processo SARIMA(p, d, q)(P, D, Q)s estende o ARIMA para incluir componentes sazonais.

\(\Phi_P(B^s) \phi_p(B) (1-B)^d (1-B^s)^D y_t = c + \Theta_Q(B^s) \theta_q(B) \epsilon_t\)

Onde: - \(B\) é o operador de retrocesso (backshift operator), \(B y_t = y_{t-1}\). - \(B^s\) é o operador de retrocesso sazonal, \(B^s y_t = y_{t-s}\). - \(\phi_p(B)\) e \(\theta_q(B)\) são os polinômios AR e MA para a parte não sazonal. - \(\Phi_P(B^s)\) e \(\Theta_Q(B^s)\) são os polinômios AR e MA para a parte sazonal. - \(d\) é a ordem de diferenciação não sazonal. - \(D\) é a ordem de diferenciação sazonal. - \(s\) é o período sazonal.

Modelos ARCH (Autoregressive Conditional Heteroskedasticity)

Um processo ARCH(q) modela a variância condicional como uma função dos erros quadrados passados.

\(y_t = \mu_t + \epsilon_t\) \(\epsilon_t = \sigma_t z_t\) \(\sigma_t^2 = \alpha_0 + \alpha_1 \epsilon_{t-1}^2 + \dots + \alpha_q \epsilon_{t-q}^2\)

Onde \(z_t\) é um ruído branco com média zero e variância unitária, e \(\alpha_i \ge 0\) para \(i=1, \dots, q\) e \(\alpha_0 > 0\).

Modelos GARCH (Generalized Autoregressive Conditional Heteroskedasticity)

Um processo GARCH(p, q) estende o ARCH para incluir termos autorregressivos na variância condicional.

\(y_t = \mu_t + \epsilon_t\) \(\epsilon_t = \sigma_t z_t\) \(\sigma_t^2 = \alpha_0 + \alpha_1 \epsilon_{t-1}^2 + \dots + \alpha_q \epsilon_{t-q}^2 + \beta_1 \sigma_{t-1}^2 + \dots + \beta_p \sigma_{t-p}^2\)

Onde \(z_t\) é um ruído branco com média zero e variância unitária, \(\alpha_i \ge 0\) para \(i=1, \dots, q\), \(\beta_j \ge 0\) para \(j=1, \dots, p\), e \(\alpha_0 > 0\). Para estacionariedade fraca, \(\sum_{i=1}^q \alpha_i + \sum_{j=1}^p \beta_j < 1\).

# Função para calcular o Cp negativo
def nCp(sigma2, estimator, X, Y):
    """
    Negative Cp statistic
    """
    n, p = X.shape
    Yhat = estimator.predict(X)
    RSS = np.sum((Y - Yhat) ** 2)
    return -(RSS + 2 * p * sigma2) / n

def preparar_dataset(df: pd.DataFrame, target: str):
    """
    Prepara X e Y a partir de um DataFrame qualquer.
    Usa ModelSpec para criar matriz de design (versão ISLP 0.4.0).
    """
    # Remove linhas com NaN no target
    df = df.dropna(subset=[target]).copy()

    # Variável resposta
    Y = np.array(df[target])

    # Cria especificação de modelo com todas as colunas menos o target
    design = ModelSpec(df.columns.drop(target)).fit(df)

    # Matriz de design
    X = design.transform(df)

    return design, X, Y

def calcular_neg_cp(df: pd.DataFrame, target: str):
    """
    Calcula estatística Cp negativa para um dataset qualquer.
    """
    # Prepara dados
    design, X, Y = preparar_dataset(df, target)

    # Ajusta modelo OLS para estimar sigma²
    sigma2 = OLS(Y, X).fit().scale

    # Cria função parcial para usar em seleção de modelos
    neg_Cp = partial(nCp, sigma2)

    return neg_Cp, design, X, Y

def ridge_lasso_regression(df: pd.DataFrame, target: str = "forward_returns",
                           test_size: float = 0.001, random_state: int = 42,
                           ridge_alpha: float = 0.5, lasso_alpha: float = 0.01):
    """
    Executa Ridge e Lasso Regression em um DataFrame e retorna métricas,
    importância das features e DataFrame reduzido com features selecionadas pelo Lasso.
    """

    # Seleciona apenas colunas numéricas e remove NaNs
    df_num = df.select_dtypes(include=np.number).dropna()

    X = df_num.drop(target, axis=1)
    y = df_num[target]
    feature_names = X.columns

    # Split
    X_train, X_test, y_train, y_test = train_test_split(
        X, y, test_size=test_size, random_state=random_state
    )

    # Escalonamento
    scaler = StandardScaler()
    X_train_scaled = scaler.fit_transform(X_train)
    X_test_scaled = scaler.transform(X_test)

    # Ridge
    ridge_model = Ridge(alpha=ridge_alpha)
    ridge_model.fit(X_train_scaled, y_train)
    ridge_preds = ridge_model.predict(X_test_scaled)
    ridge_mse = mean_squared_error(y_test, ridge_preds)
    ridge_importance = dict(zip(feature_names, ridge_model.coef_))

    # Lasso
    lasso_model = Lasso(alpha=lasso_alpha)
    lasso_model.fit(X_train_scaled, y_train)
    lasso_preds = lasso_model.predict(X_test_scaled)
    lasso_mse = mean_squared_error(y_test, lasso_preds)
    lasso_importance = dict(zip(feature_names, lasso_model.coef_))

    # Seleção de features não nulas do Lasso
    lasso_features = feature_names[lasso_model.coef_ != 0]
    df_selected_features = df[lasso_features.tolist() + [target]].copy()

    return {
        "ridge": {
            "mse": ridge_mse,
            "coefficients": ridge_model.coef_,
            "feature_importance": ridge_importance
        },
        "lasso": {
            "mse": lasso_mse,
            "coefficients": lasso_model.coef_,
            "non_zero_coeffs": np.sum(lasso_model.coef_ != 0),
            "feature_importance": lasso_importance,
            "selected_features": lasso_features.tolist(),
            "df_selected_features": df_selected_features
        }
    }

def analyze_peaks(series, num_peaks=50):
    x = np.array(series)
    N = len(x)
    fft_result = np.fft.fft(x)
    freqs = np.fft.fftfreq(N)
    mag = np.abs(fft_result)

    # detectar picos em magnitude
    peaks, _ = find_peaks(mag)

    # mapear para frequências positivas únicas (usar índice de menor abs para cada par)
    pos_map = {}
    for i in peaks:
        f = freqs[i]
        if f < 0:
            f_pos = -f
        else:
            f_pos = f
        # manter índice com maior magnitude para essa frequência positiva
        if f_pos not in pos_map or mag[i] > mag[pos_map[f_pos]]:
            pos_map[f_pos] = i

    # criar lista de (freq_positive, index, magnitude)
    pos_list = [(f, pos_map[f], mag[pos_map[f]]) for f in pos_map]
    # ordenar pela magnitude descendente
    pos_list.sort(key=lambda t: t[2], reverse=True)

    # selecionar top k
    selected = pos_list[:num_peaks]

    # calcular energia total e energia selecionada
    total_energy = np.sum(mag**2)
    selected_indices = [idx for _, idx, _ in selected] + [(-idx) % N for _, idx, _ in selected if idx != 0]
    selected_energy = np.sum(np.abs(fft_result[selected_indices])**2)

    # preparar saída legível
    result = []
    for f, idx, m in selected:
        period = 1.0 / f if f != 0 else np.inf
        result.append({'frequency': f, 'index': idx, 'magnitude': m, 'period_samples': period})

    energy_fraction = selected_energy / total_energy

    return result, energy_fraction

def reconstruct_with_peak_selection(series, num_peaks=50, freq_min=None, freq_max=None, fs=1.0):
    """
    Reconstrói a série usando os picos mais fortes da FFT, com opção de limitar por faixa de frequência.

    Parâmetros:
    - series: pandas Series ou array NumPy com os dados temporais
    - num_peaks: número máximo de picos positivos (por magnitude) a manter
    - freq_min: frequência mínima (ciclos por amostra) a considerar; se None, sem limite inferior
    - freq_max: frequência máxima (ciclos por amostra) a considerar; se None, sem limite superior
    - fs: frequência de amostragem (amostras por unidade de tempo); default 1.0

    Retorna:
    - reconstructed: série temporal reconstruída (numpy array)
    - selected_peaks_info: lista de dicionários com {'frequency', 'index', 'magnitude', 'period_samples', 'period_time'}
    - energy_fraction: fração da energia total capturada pelos componentes selecionados
    """
    x = np.array(series)
    N = len(x)
    fft_result = np.fft.fft(x)
    freqs = np.fft.fftfreq(N, d=1.0/fs)  # ciclos por unidade de tempo
    mag = np.abs(fft_result)

    # Detectar picos em magnitude sobre todo o espectro
    peaks, _ = find_peaks(mag)

    # Mapear picos para frequência positiva única (f_pos >= 0)
    pos_map = {}
    for i in peaks:
        f = freqs[i]
        f_pos = abs(f)
        # aplicar faixa de frequência se especificada
        if (freq_min is not None and f_pos < abs(freq_min)) or (freq_max is not None and f_pos > abs(freq_max)):
            continue
        # manter o índice com maior magnitude para essa frequência positiva
        if (f_pos not in pos_map) or (mag[i] > mag[pos_map[f_pos]]):
            pos_map[f_pos] = i

    # Criar lista de (f_pos, index, magnitude) e ordenar por magnitude desc
    pos_list = [(f_pos, pos_map[f_pos], mag[pos_map[f_pos]]) for f_pos in pos_map]
    pos_list.sort(key=lambda t: t[2], reverse=True)

    # Selecionar top num_peaks
    selected = pos_list[:num_peaks]

    # Construir vetor FFT filtrado preservando simetria
    filtered_fft = np.zeros_like(fft_result, dtype=complex)
    selected_indices = []
    for f_pos, idx, m in selected:
        filtered_fft[idx] = fft_result[idx]
        # índice simétrico
        if idx != 0:
            sym_idx = (-idx) % N
            filtered_fft[sym_idx] = fft_result[sym_idx]
        selected_indices.append(idx)

    # Reconstruir série
    reconstructed = np.fft.ifft(filtered_fft).real

    # Calcular energia total e energia selecionada (uso de valores complexos)
    total_energy = np.sum(np.abs(fft_result)**2)
    # incluir pares simétricos na soma de energia
    sel_idxs = []
    for _, idx, _ in selected:
        sel_idxs.append(idx)
        if idx != 0:
            sel_idxs.append((-idx) % N)
    selected_energy = np.sum(np.abs(fft_result[sel_idxs])**2) if len(sel_idxs) > 0 else 0.0
    energy_fraction = selected_energy / total_energy if total_energy > 0 else 0.0

    # Preparar info legível sobre picos
    selected_peaks_info = []
    for f_pos, idx, m in selected:
        freq = f_pos
        period_samples = (1.0 / freq) if freq != 0 else np.inf
        period_time = period_samples / fs if freq != 0 else np.inf
        selected_peaks_info.append({
            'frequency': float(freq),
            'index': int(idx),
            'magnitude': float(m),
            'period_samples': float(period_samples),
            'period_time': float(period_time)
        })

    return reconstructed, selected_peaks_info, float(energy_fraction)

class GeneticAlgorithmFeatureSelection:
    def __init__(self, n_population=50, n_generations=100, mutation_rate=0.1, crossover_rate=0.8):
        self.n_population = n_population
        self.n_generations = n_generations
        self.mutation_rate = mutation_rate
        self.crossover_rate = crossover_rate

    def initialize_population(self, n_features):
        return np.random.randint(0, 2, size=(self.n_population, n_features))

    def calculate_fitness(self, population, X, y):
        fitness = []
        tscv = TimeSeriesSplit(n_splits=3)

        for individual in population:
            selected_features = individual.astype(bool)
            if np.sum(selected_features) == 0:
                fitness.append(-np.inf)
                continue

            X_subset = X.iloc[:, selected_features]
            scores = []

            for train_idx, val_idx in tscv.split(X_subset):
                X_train, X_val = X_subset.iloc[train_idx], X_subset.iloc[val_idx]
                y_train, y_val = y.iloc[train_idx], y.iloc[val_idx]

                model = RandomForestRegressor(n_estimators=50, random_state=42)
                model.fit(X_train, y_train)
                y_pred = model.predict(X_val)
                scores.append(-mean_squared_error(y_val, y_pred))

            fitness.append(np.mean(scores))
        return np.array(fitness)

    def selection(self, population, fitness):
        selected_indices = np.random.choice(
            len(population),
            size=len(population),
            p=softmax(fitness)
        )
        return population[selected_indices]

    def crossover(self, parent1, parent2):
        if np.random.rand() < self.crossover_rate:
            point = np.random.randint(1, len(parent1)-1)
            child1 = np.concatenate([parent1[:point], parent2[point:]])
            child2 = np.concatenate([parent2[:point], parent1[point:]])
            return child1, child2
        return parent1.copy(), parent2.copy()

    def mutation(self, individual):
        for i in range(len(individual)):
            if np.random.rand() < self.mutation_rate:
                individual[i] = 1 - individual[i]
        return individual

    def evolve(self, X, y):
        n_features = X.shape[1]
        population = self.initialize_population(n_features)
        best_individual = None
        best_fitness = -np.inf

        for generation in range(self.n_generations):
            fitness = self.calculate_fitness(population, X, y)

            # Atualizar melhor indivíduo
            current_best_idx = np.argmax(fitness)
            if fitness[current_best_idx] > best_fitness:
                best_fitness = fitness[current_best_idx]
                best_individual = population[current_best_idx]

            # Operações genéticas
            selected_population = self.selection(population, fitness)
            next_population = []

            for i in range(0, len(selected_population), 2):
                parent1 = selected_population[i]
                parent2 = selected_population[i+1] if i+1 < len(selected_population) else selected_population[0]
                child1, child2 = self.crossover(parent1, parent2)
                next_population.extend([self.mutation(child1), self.mutation(child2)])

            population = np.array(next_population)[:self.n_population]

            if generation % 20 == 0:
                print(f"Geração {generation}: Melhor Fitness = {best_fitness:.4f}")

        return best_individual

# -----------------------
# 2) Feature engineering
# -----------------------
def make_lag_features(df, col, lags=[1,2,3,5,10]):
    for l in lags:
        df[f"{col}_lag{l}"] = df[col].shift(l)
    return df

def make_diff_features(df, col, lags=[1,2,3]):
    for l in lags:
        df[f"{col}_diff{l}"] = df[col] - df[col].shift(l)
    return df

def make_rolling_features(df, col, windows=[3,5,10]):
    for w in windows:
        df[f"{col}_ma{w}"] = df[col].rolling(window=w, min_periods=1).mean()
        df[f"{col}_std{w}"] = df[col].rolling(window=w, min_periods=1).std().fillna(0)
    return df

df = make_lag_features(df, target_col, lags=[1,2,3,5,10])
df = make_diff_features(df, target_col, lags=[1,2,3])
df = make_rolling_features(df, target_col, windows=[3,5,10])

# Alvo binário: 1 se subir no próximo dia, 0 caso contrário
df["target_next_up"] = (df[target_col].shift(-1) > df[target_col]).astype(int)

# Remove linhas com NaNs gerados pela defasagem

# -----------------------
# 3) Walk-forward CV setup
# -----------------------
def walk_forward_splits(n_samples, n_splits=5, test_size=None):
    # Divides into n_splits sequential folds with expanding train and fixed/rolling test
    if test_size is None:
        test_size = n_samples // (n_splits + 1)
    splits = []
    start = 0
    for i in range(n_splits):
        train_end = start + (i + 1) * test_size
        test_start = train_end
        test_end = min(test_start + test_size, n_samples)
        if test_end <= test_start:
            break
        splits.append((slice(0, train_end), slice(test_start, test_end)))
    return splits

n_samples = len(df)
splits = walk_forward_splits(n_samples, n_splits=5)

# -----------------------
# 4) Helper: train PySR and evaluate
# -----------------------
def train_pysr(X_train, y_train, X_val, timeout=60, n_iters=40, procs=1, extra_symbols=None):
    # PySRRegressor returns a symbolic continuous predictor; we'll threshold it later
    custom_ops = ["+", "-", "*", "/","sin","cos","exp","log","sqrt"]
    if extra_symbols:
        custom_ops += extra_symbols
    model = PySRRegressor(
        unary_operators=["sin", "cos", "exp", "log", "sqrt"],
        binary_operators=["+", "-", "*", "/"],
        model_selection="best",        # choose best by loss+complexity on internal pareto
        niterations=n_iters,
        timeout=timeout,              # seconds
        # Use loss appropriate for continuous regression; we'll evaluate classification externally
        loss="loss(x, y) = (x - y)**2",
        popsize=20,
        procs=procs,
        random_state=0,
        maxsize=20,
    )
    model.fit(X_train.values, y_train.values)
    preds_val = model.predict(X_val.values)
    return model, preds_val

    thresh_grid = np.linspace(val_scores.min(), val_scores.max(), 101)
    best_t = 0.5
    best_f1 = -1
    for t in thresh_grid:
        preds = (val_scores > t).astype(int)
        f1 = f1_score(y_val, preds)
        if f1 > best_f1:
            best_f1 = f1
            best_t = t

def ta_dashboard(df=None, asset='AAPL', start='2018-01-01', end='2018-12-31'):
    """
    Cria um painel interativo para análise técnica com Bollinger Bands, MACD e RSI.

    Parâmetros:
    -----------
    df : pd.DataFrame, opcional
        DataFrame com colunas ['Open','High','Low','Close','Volume'].
        Se None, os dados serão baixados do Yahoo Finance.
    asset : str
        Símbolo do ativo (ex: 'AAPL', 'MSFT', 'TSLA', etc).
    start : str
        Data inicial no formato 'YYYY-MM-DD'.
    end : str
        Data final no formato 'YYYY-MM-DD'.
    """
    import pandas as pd
    import yfinance as yf
    import cufflinks as cf
    import ipywidgets as wd
    from ipywidgets import interactive_output, HBox, VBox, Label
    from IPython.display import display
    import plotly.graph_objects as go
    from plotly.subplots import make_subplots
    from ta.volatility import BollingerBands
    from ta.trend import MACD
    from ta.momentum import RSIIndicator

    # Configurar cufflinks
    cf.go_offline()

    # -------------------------------
    # 🔹 Carrega e prepara dados
    # -------------------------------
    def prepare_dataframe(df, asset, start, end):
        """Prepara o DataFrame no formato correto para análise técnica"""
        if df is None:
            # Baixar dados sem auto_adjust para ter controle total
            try:
                df_raw = yf.download(asset, start=start, end=end, progress=False)

                if df_raw.empty:
                    print(f"⚠️ Não foi possível baixar dados para {asset} no período {start} a {end}. Verifique o ticker e o intervalo de datas.")
                    return pd.DataFrame(columns=['Open','High','Low','Close','Volume'])

                # Handle case where yfinance returns a Series for single column requests (e.g., 'Adj Close')
                if isinstance(df_raw, pd.Series):
                    df_raw = df_raw.to_frame()

                # Ensure required columns exist before accessing
                required_cols_yf = ['Open', 'High', 'Low', 'Close', 'Volume']
                existing_cols_yf = [col for col in required_cols_yf if col in df_raw.columns]

                if len(existing_cols_yf) < len(required_cols_yf):
                     print(f"⚠️ Dados baixados para {asset} não contêm todas as colunas esperadas: {required_cols_yf}. Colunas encontradas: {list(df_raw.columns)}")
                     return pd.DataFrame(columns=required_cols_yf)


                # Criar DataFrame com estrutura explícita
                df_clean = pd.DataFrame({
                    'Open': df_raw['Open'],
                    'High': df_raw['High'],
                    'Low': df_raw['Low'],
                    'Close': df_raw['Close'],
                    'Volume': df_raw['Volume']
                })

                # Usar Adjusted Close se disponível, caso contrário usar Close
                if 'Adj Close' in df_raw.columns:
                    df_clean['Close'] = df_raw['Adj Close']

            except Exception as e:
                print(f"❌ Erro ao baixar dados para {asset}: {e}")
                return pd.DataFrame(columns=['Open','High','Low','Close','Volume'])

        else:
            df_clean = df.copy()
            # Garantir que o índice é datetime
            df_clean.index = pd.to_datetime(df_clean.index)

            # Verificar e renomear colunas se necessário
            column_mapping = {
                'open': 'Open', 'OPEN': 'Open',
                'high': 'High', 'HIGH': 'High',
                'low': 'Low', 'LOW': 'Low',
                'close': 'Close', 'CLOSE': 'Close',
                'volume': 'Volume', 'VOLUME': 'Volume',
                'adj close': 'Close', 'Adj Close': 'Close'
            }

            for old_col, new_col in column_mapping.items():
                if old_col in df_clean.columns:
                    df_clean[new_col] = df_clean[old_col]

            # Manter apenas as colunas necessárias
            required_cols = ['Open', 'High', 'Low', 'Close', 'Volume']
            # Check if required columns exist before selecting
            existing_required_cols = [col for col in required_cols if col in df_clean.columns]
            if len(existing_required_cols) < len(required_cols):
                 print(f"⚠️ DataFrame fornecido não contém todas as colunas necessárias: {required_cols}. Colunas encontradas: {list(df_clean.columns)}")
                 return pd.DataFrame(columns=required_cols)

            df_clean = df_clean[existing_required_cols].copy()


        return df_clean.dropna()

    # Preparar dados iniciais
    df_processed = prepare_dataframe(df, asset, start, end)

    # -------------------------------
    # 🔹 Parâmetros interativos
    # -------------------------------
    stocks = ['AAPL', 'MSFT', 'GOOGL', 'TSLA', 'META', 'NVDA', 'AMZN']
    indicators = ['Bollinger Bands', 'MACD', 'RSI']

    stocks_selector = wd.Dropdown(options=stocks, value=asset, description='Ativo:')
    indicator_selector = wd.SelectMultiple(
        description='Indicadores:',
        options=indicators,
        value=[indicators[0]]
    )

    start_date_selector = wd.DatePicker(description='Início:', value=pd.to_datetime(start))
    end_date_selector = wd.DatePicker(description='Fim:', value=pd.to_datetime(end))

    # ---- Bollinger Bands ----
    bb_label = wd.Label('📊 Bollinger Bands')
    n_param = wd.IntSlider(value=20, min=5, max=50, step=1, description='N:')
    k_param = wd.FloatSlider(value=2.0, min=0.5, max=4.0, step=0.1, description='k:')

    # ---- MACD ----
    macd_label = wd.Label('📈 MACD')
    macd_fast = wd.IntSlider(value=12, min=2, max=50, step=1, description='Rápida:')
    macd_slow = wd.IntSlider(value=26, min=2, max=50, step=1, description='Lenta:')
    macd_signal = wd.IntSlider(value=9, min=2, max=50, step=1, description='Sinal:')

    # ---- RSI ----
    rsi_label = wd.Label('📉 RSI')
    rsi_periods = wd.IntSlider(value=14, min=2, max=50, step=1, description='Períodos:')
    rsi_upper = wd.IntSlider(value=70, min=50, max=100, step=1, description='Limite Sup.:')
    rsi_lower = wd.IntSlider(value=30, min=1, max=50, step=1, description='Limite Inf.:')

    # -------------------------------
    # 🔹 Função alternativa para gráfico (usando Plotly diretamente)
    # -------------------------------
    def plot_dashboard(asset, indicator, start_date, end_date,
                       bb_k, bb_n, macd_fast, macd_slow, macd_signal,
                       rsi_periods, rsi_upper, rsi_lower):

        try:
            # Usar dados processados ou baixar novos se o ativo mudou ou data mudou
            current_asset = stocks_selector.value
            current_start = start_date_selector.value
            current_end = end_date_selector.value

            if df is None or current_asset != asset or pd.to_datetime(current_start) != pd.to_datetime(start_date) or pd.to_datetime(current_end) != pd.to_datetime(end_date):
                 df_plot = prepare_dataframe(None, current_asset, current_start, current_end)
            else:
                 df_plot = df_processed.copy()


            if df_plot.empty:
                print("⚠️ Nenhum dado encontrado ou selecionado para o período/ativo.")
                return


            # Filtrar por data (redundante se prepare_dataframe já filtrou, mas seguro)
            mask = (df_plot.index >= pd.to_datetime(start_date)) & (df_plot.index <= pd.to_datetime(end_date))
            df_plot = df_plot.loc[mask]

            if df_plot.empty:
                print("⚠️ Nenhum dado encontrado para o período selecionado após filtragem.")
                return


            # Calcular indicadores manualmente
            # from ta.volatility import BollingerBands # Imported at top
            # from ta.trend import MACD # Imported at top
            # from ta.momentum import RSIIndicator # Imported at top

            # Criar figura com subplots
            num_indicator_plots = len(indicator)
            num_plots = 1 + num_indicator_plots  # Preço + indicadores
            row_heights = [0.6] + [0.4/num_indicator_plots] * num_indicator_plots if num_indicator_plots > 0 else [1.0]

            fig = make_subplots(
                rows=num_plots, cols=1,
                shared_xaxes=True,
                vertical_spacing=0.05,
                subplot_titles=['Preço'] + list(indicator),
                row_heights=row_heights
            )

            # Gráfico de preço
            fig.add_trace(go.Candlestick(
                x=df_plot.index,
                open=df_plot['Open'],
                high=df_plot['High'],
                low=df_plot['Low'],
                close=df_plot['Close'],
                name='Price'
            ), row=1, col=1)

            # Bollinger Bands
            if 'Bollinger Bands' in indicator:
                bb_indicator = BollingerBands(close=df_plot['Close'], window=bb_n, window_dev=bb_k)
                df_plot['BB_Upper'] = bb_indicator.bollinger_hband()
                df_plot['BB_Lower'] = bb_indicator.bollinger_lband()
                df_plot['BB_Middle'] = bb_indicator.bollinger_mavg()
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['BB_Upper'],
                    line=dict(color='rgba(255,0,0,0.5)', width=1),
                    name='BB Upper', legendgroup='BB', showlegend=True
                ), row=1, col=1)
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['BB_Lower'],
                    line=dict(color='rgba(0,255,0,0.5)', width=1),
                    name='BB Lower', legendgroup='BB', showlegend=True,
                    fill='tonexty', fillcolor='rgba(0,255,0,0.1)' # Add fill
                ), row=1, col=1)
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['BB_Middle'],
                    line=dict(color='rgba(0,0,255,0.5)', width=1),
                    name='BB Middle', legendgroup='BB', showlegend=True
                ), row=1, col=1)

            # MACD
            if 'MACD' in indicator:
                row_num = 1 + list(indicator).index('MACD') + 1
                macd_indicator = MACD(close=df_plot['Close'], window_fast=macd_fast,
                                    window_slow=macd_slow, window_sign=macd_signal)
                df_plot['MACD'] = macd_indicator.macd()
                df_plot['MACD_Signal'] = macd_indicator.macd_signal()
                df_plot['MACD_Histogram'] = macd_indicator.macd_diff()
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['MACD'],
                    line=dict(color='blue', width=2),
                    name='MACD', legendgroup='MACD', showlegend=True
                ), row=row_num, col=1)
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['MACD_Signal'],
                    line=dict(color='red', width=2),
                    name='MACD Signal', legendgroup='MACD', showlegend=True
                ), row=row_num, col=1)
                fig.add_trace(go.Bar(
                    x=df_plot.index, y=df_plot['MACD_Histogram'],
                    marker_color='gray',
                    name='MACD Histogram', legendgroup='MACD', showlegend=True
                ), row=row_num, col=1)

            # RSI
            if 'RSI' in indicator:
                row_num = 1 + list(indicator).index('RSI') + 1
                rsi_indicator = RSIIndicator(close=df_plot['Close'], window=rsi_periods)
                df_plot['RSI'] = rsi_indicator.rsi()
                fig.add_trace(go.Scatter(
                    x=df_plot.index, y=df_plot['RSI'],
                    line=dict(color='purple', width=2),
                    name='RSI', legendgroup='RSI', showlegend=True
                ), row=row_num, col=1)
                # Linhas de sobrecompra/sobrevenda
                fig.add_hline(y=rsi_upper, line_dash="dash", line_color="red", row=row_num, col=1)
                fig.add_hline(y=rsi_lower, line_dash="dash", line_color="green", row=row_num, col=1)

            fig.update_layout(
                title=f'📊 Painel Técnico - {current_asset}',
                height=800,
                showlegend=True,
                xaxis_rangeslider_visible=False # Disable the default range slider
            )

            fig.show()

        except Exception as e:
            print(f"❌ Erro ao gerar gráfico: {str(e)}")
            import traceback
            traceback.print_exc()


    # -------------------------------
    # 🔹 Layout interativo
    # -------------------------------
    main_box = VBox([
        Label('🎯 Parâmetros Principais'),
        stocks_selector,
        indicator_selector,
        start_date_selector,
        end_date_selector
    ], layout=wd.Layout(margin='10px', padding='10px', width='300px'))

    bb_box = VBox([bb_label, n_param, k_param], layout=wd.Layout(margin='5px'))
    macd_box = VBox([macd_label, macd_fast, macd_slow, macd_signal], layout=wd.Layout(margin='5px'))
    rsi_box = VBox([rsi_label, rsi_periods, rsi_upper, rsi_lower], layout=wd.Layout(margin='5px'))

    indicators_box = HBox([
        VBox([Label('📈 Indicadores Técnicos'), bb_box, macd_box, rsi_box])
    ], layout=wd.Layout(margin='10px'))

    controls = {
        'asset': stocks_selector,
        'indicator': indicator_selector,
        'start_date': start_date_selector,
        'end_date': end_date_selector,
        'bb_k': k_param,
        'bb_n': n_param,
        'macd_fast': macd_fast,
        'macd_slow': macd_slow,
        'macd_signal': macd_signal,
        'rsi_periods': rsi_periods,
        'rsi_upper': rsi_upper,
        'rsi_lower': rsi_lower
    }

    ui = VBox([
        HBox([main_box, indicators_box]),
    ], layout=wd.Layout(
        border='1px solid gray',
        padding='15px',
        margin='10px'
    ))

    out = interactive_output(plot_dashboard, controls)

    display(ui, out)

import yfinance as yf

def run_time_series_analysis(df=None, asset='AAPL', start='2018-01-01', end='2023-12-31', column='Close'):
    import pandas as pd
    import numpy as np
    import matplotlib.pyplot as plt
    import seaborn as sns
    #import yfinance as yf
    from statsmodels.tsa.seasonal import seasonal_decompose
    from statsmodels.tsa.stattools import adfuller, kpss
    from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
    from statsmodels.tsa.holtwinters import ExponentialSmoothing
    from statsmodels.tsa.arima.model import ARIMA
    import warnings
    warnings.filterwarnings("ignore")

    sns.set(style="whitegrid", rc={'figure.figsize': (12, 6)})

    # -------------------------------
    # 1️⃣ Carregamento dos dados
    # -------------------------------
    if df is None:
        print(f"📥 Baixando dados de {asset} de {start} até {end} ...")
        df = yf.download(asset, start=start, end=end, progress=False, auto_adjust=True)
    df = df.copy()
    df = df.asfreq('B')  # frequência diária (dias úteis)
    df[column] = df[column].interpolate()

    print(f"\n✅ Dados carregados: {len(df)} registros entre {df.index.min().date()} e {df.index.max().date()}")

    series = df[column]

    # -------------------------------
    # 2️⃣ Visualização da série original
    # -------------------------------

    plt.plot(series, color='darkblue')
    plt.title(f'Série Temporal - {asset} ({column})')
    plt.xlabel('Data')
    plt.ylabel(column)
    plt.show()

    # -------------------------------
    # 3️⃣ Decomposição da série
    # -------------------------------
    print("\n🔍 Decomposição da Série (Tendência, Sazonalidade e Resíduo):")
    decomposition = seasonal_decompose(series, model='additive', period=252)
    decomposition.plot()
    plt.show()

    # -------------------------------
    # 4️⃣ Testes de estacionariedade
    # -------------------------------
    def adf_test(series):
        result = adfuller(series, autolag='AIC')
        return {'ADF': result[0], 'p-valor': result[1], 'Critério 5%': result[4]['5%']}

    def kpss_test(series):
        result = kpss(series, regression='c', nlags='auto')
        return {'KPSS': result[0], 'p-valor': result[1], 'Critério 5%': result[3]['5%']}

    adf_result = adf_test(series)
    kpss_result = kpss_test(series)

    print("\n📈 Teste ADF (Dickey-Fuller aumentado):")

    plt.title(f'Previsão ARIMA(1,{d},1)')

def run_multi_factor_analysis(
    assets=['AMZN'],
    benchmark='^GSPC',
    start='2014-01-01',
    end='2018-12-31',
    weights=None,
    ff_csv_path=None,       # optional: path to 'Research_Data_Factors.csv' if you have it
    rolling_window=60
):
    """
    Executa CAPM, Fama-French 3-factor, rolling 3-factor on a portfolio,
    and 4- & 5-factor models (incl. momentum).

    Parâmetros:
    - assets: list of ticker strings (ex: ['AMZN','AAPL'])
    - benchmark: mercado (ex: '^GSPC')
    - start, end: strings 'YYYY-MM-DD'
    - weights: list of portfolio weights (same length as assets). If None, equal weights.
    - ff_csv_path: opcional, caminho para Research_Data_Factors.csv (se preferir usar arquivo local)
    - rolling_window: janela (em meses) para o modelo rolling
    """
    import pandas as pd
    import numpy as np
    import yfinance as yf
    import matplotlib.pyplot as plt
    import seaborn as sns
    import statsmodels.api as sm
    import statsmodels.formula.api as smf
    from pandas_datareader import data as web
    import warnings
    warnings.filterwarnings('ignore')
    sns.set(style='whitegrid')
    plt.rcParams['figure.figsize'] = (10,5)

    # ------------------------------
    # Helper prints
    # ------------------------------
    def banner(msg):
        print('\n' + '='*8 + ' ' + msg + ' ' + '='*8)

    # ------------------------------
    # 1) CAPM: Beta (covariance) + OLS
    # ------------------------------
    banner('CAPM - Beta and OLS')
    tickers = list(set(assets + [benchmark]))
    print(f'Downloading tickers: {tickers} from {start} to {end} ...')
    raw = yf.download(tickers, start=start, end=end, progress=False)
    if 'Adj Close' in raw:
        adj = raw['Adj Close']
    else:
        # fallback if yfinance structure differs
        adj = raw

    # keep only the tickers we want
    adj = adj[tickers].dropna(how='all')
    # monthly simple returns
    monthly = adj.resample('M').last().pct_change().dropna()
    # prepare X for CAPM
    asset_col = assets[0] if len(assets)==1 else assets[0]
    if asset_col not in monthly.columns:
        raise ValueError(f'Ativo {asset_col} não está nos dados baixados.')
    X = monthly[[asset_col, benchmark]].rename(columns={asset_col:'asset', benchmark:'market'})
    X = X.dropna()
    print(f'Usando {len(X)} observações mensais para CAPM.')

    # covariance approach
    cov = X.cov().iloc[0,1]
    var_mkt = X.market.var()
    beta_cov = cov / var_mkt
    print(f'Beta (covariance method) = {beta_cov:.4f}')

    # OLS regression
    y = X['asset']
    X_reg = sm.add_constant(X['market'])
    capm_model = sm.OLS(y, X_reg).fit()
    print(capm_model.summary())

    # ------------------------------
    # 2) Fama-French 3-factor model
    # ------------------------------
    banner('Fama-French Three-Factor')
    # load factors: try pandas_datareader first, else use csv path if given
    try:
        ff = web.DataReader('F-F_Research_Data_Factors', 'famafrench', start=start)[0]
        ff.index = ff.index.map(lambda d: d.strftime('%Y-%m'))
        ff = ff.div(100)  # convert to decimal
        print('Loaded Fama-French factors via pandas_datareader (famafrench).')
    except Exception as e:
        if ff_csv_path is None:
            print('Não foi possível obter Fama-French via pandas_datareader. '
                  'Se você tiver o CSV (Research_Data_Factors.csv), passe ff_csv_path.')
            raise e
        else:
            print('Carregando fatores do arquivo CSV fornecido...')
            ff_raw = pd.read_csv(ff_csv_path, skiprows=3)
            # find start of annual block to slice monthly
            marker = ' Annual Factors: January-December '
            try:
                start_of_annual = ff_raw[ff_raw.iloc[:,0]==marker].index[0]
                ff_raw = ff_raw.iloc[:start_of_annual]
            except Exception:
                pass
            ff_raw.columns = ['date','mkt','smb','hml','rf'] + list(ff_raw.columns[5:])
            ff_raw['date'] = pd.to_datetime(ff_raw['date'], format='%Y%m').dt.strftime('%Y-%m')
            ff = ff_raw.set_index('date')[['mkt','smb','hml','rf']].apply(pd.to_numeric, errors='coerce').div(100)
            print('Fatores carregados do CSV.')

    # asset returns (monthly) for single risky asset (first asset in list)
    asset_df = yf.download(assets[0], start=start, end=end, progress=False)
    if 'Adj Close' in asset_df:
        asset_df = asset_df['Adj Close']
    rtn = asset_df.resample('M').last().pct_change().dropna()
    rtn.index = rtn.index.strftime('%Y-%m')
    y_ff = rtn.rename('rtn').to_frame()
    # join
    ff_data = ff.join(y_ff, how='inner')
    ff_data['excess_rtn'] = ff_data.rtn - ff_data.rf
    print(f'FF sample size: {len(ff_data)} months after join.')

    # regression
    ff_model = smf.ols('excess_rtn ~ mkt + smb + hml', data=ff_data).fit()
    print(ff_model.summary())

    # ------------------------------
    # 3) Rolling three-factor on portfolio
    # ------------------------------
    if weights is None:
        weights = [1.0/len(assets)]*len(assets)
    if len(weights) != len(assets):
        raise ValueError('weights deve ter o mesmo tamanho que assets.')
    banner('Rolling Three-Factor on Portfolio')
    # download price series for all assets
    asset_prices = yf.download(assets, start=start, end=end, progress=False)['Adj Close']
    # monthly returns
    asset_monthly = asset_prices.resample('M').last().pct_change().dropna()
    asset_monthly.index = asset_monthly.index.strftime('%Y-%m')
    # portfolio returns
    portfolio_vals = (asset_monthly[assets].values @ np.array(weights))
    portf = pd.Series(portfolio_vals, index=asset_monthly.index, name='portfolio_returns').to_frame()

    # get FF factors via pandas_datareader (if not already)
    try:
        df_three_factor = web.DataReader('F-F_Research_Data_Factors', 'famafrench', start=start)[0].div(100)
        df_three_factor.index = df_three_factor.index.map(lambda d: d.strftime('%Y-%m'))
    except Exception:
        df_three_factor = ff  # fallback to previously loaded

    # merge
    portf_df = portf.join(df_three_factor, how='inner')
    portf_df['portf_ex_rtn'] = portf_df.portfolio_returns - portf_df['RF'] if 'RF' in portf_df.columns else portf_df.portfolio_returns - portf_df['rf']
    # normalize column names
    if 'RF' in portf_df.columns and 'rf' not in portf_df.columns:
        portf_df = portf_df.rename(columns={'RF':'rf'})
    portf_df = portf_df.rename(columns={portf_df.columns[1]:'mkt'}) if 'mkt' not in portf_df.columns else portf_df

    # ensure columns exist
    required_cols = ['portf_ex_rtn','mkt','smb','hml']
    for c in required_cols:
        if c not in portf_df.columns:
            raise RuntimeError(f'Coluna esperada "{c}" não encontrada em portf_df. Colunas existentes: {list(portf_df.columns)}')

    # rolling regression
    def rolling_factor_model(input_data, formula, window_size):
        coeffs = []
        idx = []
        for start_i in range(len(input_data) - window_size + 1):
            end_i = start_i + window_size
            sub = input_data.iloc[start_i:end_i]
            model = smf.ols(formula=formula, data=sub).fit()
            coeffs.append(model.params)
            idx.append(input_data.index[end_i-1])
        coeffs_df = pd.DataFrame(coeffs, index=idx)
        return coeffs_df

    MODEL_FORMULA = 'portf_ex_rtn ~ mkt + smb + hml'
    # adapt RF name if necessary
    if 'rf' not in portf_df.columns:
        portf_df['rf'] = 0.0
    results_df = rolling_factor_model(portf_df[['portf_ex_rtn','mkt','smb','hml']], MODEL_FORMULA, window_size=rolling_window)
    print(f'Rolling regression produced {len(results_df)} coefficient rows (window={rolling_window}).')
    results_df.plot(title='Rolling Fama-French Three-Factor Coefficients')
    plt.show()

    # ------------------------------
    # 4) Four- and Five-Factor models (including momentum)
    # ------------------------------
    banner('Four-Factor (with MOM) & Five-Factor Models')
    # download necessary factor sets via pandas_datareader
    try:
        df_three = web.DataReader('F-F_Research_Data_Factors', 'famafrench', start=start)[0]
        df_three.index = df_three.index.map(lambda d: d.strftime('%Y-%m'))
        df_three = df_three.div(100)
        df_mom = web.DataReader('F-F_Momentum_Factor', 'famafrench', start=start)[0]
        df_mom.index = df_mom.index.map(lambda d: d.strftime('%Y-%m'))
        df_mom = df_mom.div(100)
        df_five = web.DataReader('F-F_Research_Data_5_Factors_2x3', 'famafrench', start=start)[0]
        df_five.index = df_five.index.map(lambda d: d.strftime('%Y-%m'))
        df_five = df_five.div(100)
        print('Loaded 3-, momentum-, and 5-factor datasets via pandas_datareader.')
    except Exception as e:
        print('Falha ao carregar alguns conjuntos via pandas_datareader, tentando reutilizar dados carregados anteriormente.')
        df_three = df_three_factor if 'df_three_factor' in locals() else ff
        df_mom = pd.DataFrame()  # may be empty -> check later
        df_five = pd.DataFrame()

    # prepare asset returns (single asset case)
    asset_prices_single = yf.download(assets[0], start=start, end=end, progress=False)
    if 'Adj Close' in asset_prices_single:
        asset_prices_single = asset_prices_single['Adj Close']
    y = asset_prices_single.resample('M').last().pct_change().dropna()
    y.index = y.index.strftime('%Y-%m')
    y = y.rename('return').to_frame()

    # Four-factor: join df_three + momentum + y
    if not df_mom.empty:
        four_factor_data = df_three.join(df_mom).join(y, how='inner')
        # rename columns to expected names if necessary
        # the typical df_three has columns ['Mkt-RF','SMB','HML','RF'] and df_mom ['MOM']
        # adapt names to mkt, smb, hml, rf, mom
        colmap = {}
        for c in four_factor_data.columns:
            lc = c.lower()
            if 'mkt' in lc and 'rf' in lc and 'mkt-rf' in lc:
                colmap[c] = 'mkt'
            elif lc.strip() in ['smb','hml','rf','mom']:
                colmap[c] = lc.strip()
        four_factor_data = four_factor_data.rename(columns=colmap)
        # make sure divide by 100 already handled (we used div(100))
    else:
        four_factor_data = pd.DataFrame()

    # Five-factor
    if not df_five.empty:
        five_factor_data = df_five.join(y, how='inner')
        # rename typical columns to mkt,smb,hml,rmw,cma,rf

def run_vol_forecast_pipeline(
    assets=['GOOG'],
    start='2015-01-01',
    end='2020-12-31',
    freq='D',                 # frequência de index (D = diário). Não altera returns calculation.
    univariate_mean_zero=True,# se True usa mean='Zero' nos modelos univariados univariados (como no seu código)
    garch_order=(1,1),
    arch_order=(1,0,0),
    horizon_1step=True,       # calcula previsão 1-step ahead de variâncias (para CCC)
    use_dcc_r=False,          # tenta rodar DCC-GARCH em R via rpy2 (requer R + rmgarch)
    dcc_n_ahead=5,
    plot_results=True,
    random_seed=42
):
    """
    Pipeline único para:
      - ARCH(1) (univariado)
      - GARCH(1,1) (univariado)
      - CCC-GARCH (aproximação via ajuste de univariados + correlação constante)
      - (opcional) DCC-GARCH via R ('rmgarch') se use_dcc_r=True

    Parâmetros:
      assets: lista de tickers (ex: ['GOOG'] ou ['GOOG','MSFT','AAPL'])
      start, end: intervalo (YYYY-MM-DD)
      univariate_mean_zero: se True, mean='Zero' (seguindo seu exemplo)
      garch_order: tupla (p,q) para GARCH(p,q)
      arch_order: tupla (p,o,q) para ARCH specification (usado se pedir ARCH)
      horizon_1step: calcula 1-step ahead forecast das variâncias para CCC
      use_dcc_r: tenta executar DCC-GARCH em R via rpy2 (requer rmgarch no R)
      dcc_n_ahead: horizonte para forecast DCC (se use_dcc_r=True)
      plot_results: se True, gera os gráficos principais
      random_seed: semente para reprodução
    Retorna:
      dict com objetos: data, returns, modelos univariados, coeffs_df, cond_vol_df, std_resids_df, R, H, dcc_forecast (se disponível)
    """
    import warnings
    warnings.filterwarnings('ignore')
    import numpy as np
    import pandas as pd
    import yfinance as yf
    from arch import arch_model
    import matplotlib.pyplot as plt
    import seaborn as sns
    import datetime
    np.random.seed(random_seed)
    sns.set(style='whitegrid')
    plt.rcParams['figure.figsize'] = [10, 5]
    plt.rcParams['figure.dpi'] = 200

    results = {}
    # ---------------------------
    # 1) Download data
    # ---------------------------
    tickers = list(assets)
    if len(tickers) == 0:
        raise ValueError("Forneça pelo menos 1 ticker em 'assets'.")
    raw = yf.download(tickers, start=start, end=end, progress=False, adjusted=True)
    # yfinance returns multiindex when multiple tickers; get 'Adj Close' if available
    if 'Adj Close' in raw:
        adj = raw['Adj Close']
    else:
        # single ticker case may return simple DataFrame with 'Adj Close' column
        if isinstance(raw, pd.DataFrame) and 'Adj Close' in raw.columns:
            adj = raw['Adj Close'].to_frame() if raw['Adj Close'].ndim==1 else raw['Adj Close']
        else:
            adj = raw
    # ensure DataFrame
    if isinstance(adj, pd.Series):
        adj = adj.to_frame(name=tickers[0])
    adj = adj.dropna(how='all')
    results['adj_close'] = adj.copy()
    # compute simple returns (not percent)
    returns = adj.pct_change().dropna()
    # convert to percent scale (like in your notebook)
    returns_pct = 100 * returns
    results['returns'] = returns_pct

    if plot_results:
        ax = returns_pct.plot(title=f'Returns ({start} to {end})')
        ax.set_ylabel('Returns (%)')
        plt.tight_layout()
        plt.show()

    # ---------------------------
    # 2) If single asset: ARCH and GARCH
    # ---------------------------
    single_asset_models = {}
    if len(tickers) == 1:
        asset = tickers[0]
        series = returns_pct[asset].dropna()
        # ARCH
        p, o, q = arch_order if len(arch_order)==3 else (arch_order[0], 0, 0)
        try:
            arch_spec = arch_model(series, mean='Zero' if univariate_mean_zero else 'Constant',
                                   vol='ARCH', p=p, o=o, q=q)
            arch_fit = arch_spec.fit(disp='off')
            single_asset_models['arch'] = arch_fit
            if plot_results:
                print(f"--- ARCH({p},{o},{q}) fit summary for {asset} ---")
                print(arch_fit.summary())
                arch_fit.plot(annualize='D')
                plt.tight_layout()
                plt.show()
        except Exception as e:
            single_asset_models['arch_error'] = str(e)
            print("Falha ao ajustar ARCH:", e)

        # GARCH
        p_g, q_g = garch_order
        try:
            garch_spec = arch_model(series, mean='Zero' if univariate_mean_zero else 'Constant',
                                    vol='GARCH', p=p_g, o=0, q=q_g)
            garch_fit = garch_spec.fit(disp='off')
            single_asset_models['garch'] = garch_fit
            if plot_results:
                print(f"--- GARCH({p_g},{q_g}) fit summary for {asset} ---")
                print(garch_fit.summary())
                garch_fit.plot(annualize='D')
                plt.tight_layout()
                plt.show()
        except Exception as e:
            single_asset_models['garch_error'] = str(e)
            print("Falha ao ajustar GARCH:", e)
    results['single_asset_models'] = single_asset_models

    # ---------------------------
    # 3) Multivariate: CCC-GARCH approximation
    #    Fit univariate GARCH for each asset, compute std residuals, R, and H (1-step)
    # ---------------------------
    if len(tickers) >= 1:
        coeffs = []
        cond_vol = []
        std_resids = []
        fitted_models = []
        for asset in tickers:
            ser = returns_pct[asset].dropna()
            # use mean='Constant' for multivariate (as in your example)
            try:
                m = arch_model(ser, mean='Constant', vol='GARCH', p=garch_order[0], q=garch_order[1])
                fit = m.fit(disp='off', update_freq=0)
                fitted_models.append(fit)
                coeffs.append(fit.params)
                cond_vol.append(fit.conditional_volatility)
                # standardized residuals = resid / cond_vol
                std_resids.append((fit.resid / fit.conditional_volatility).dropna())
            except Exception as e:
                # store exception
                fitted_models.append(None)
                coeffs.append({'error': str(e)})
                cond_vol.append(pd.Series(dtype=float))
                std_resids.append(pd.Series(dtype=float))
                print(f"Falha ao ajustar GARCH para {asset}: {e}")

        # convert lists to DataFrames aligned by index
        # coeffs_df: parameters per asset
        try:
            coeffs_df = pd.DataFrame(coeffs, index=tickers)
        except Exception:
            # some entries may be dicts with error
            coeffs_df = pd.DataFrame([c if isinstance(c, dict) else c.to_dict() for c in coeffs], index=tickers)
        # cond_vol_df: align by dates
        try:
            cond_vol_df = pd.concat([cv.rename(col) if isinstance(cv, pd.Series) else pd.Series(dtype=float)
                                     for cv, col in zip(cond_vol, tickers)], axis=1)
            cond_vol_df.columns = tickers
        except Exception:
            cond_vol_df = pd.DataFrame()
        # std_resids_df: need to align by index (dates)
        try:
            std_resids_df = pd.concat([sr.rename(col) if isinstance(sr, pd.Series) else pd.Series(dtype=float)
                                      for sr, col in zip(std_resids, tickers)], axis=1)
            std_resids_df.columns = tickers
        except Exception:
            std_resids_df = pd.DataFrame()

        results['coeffs_df'] = coeffs_df
        results['cond_vol_df'] = cond_vol_df
        results['std_resids_df'] = std_resids_df
        results['fitted_models'] = dict(zip(tickers, fitted_models))

        if plot_results:
            print("\nCoeficientes (por ativo):")
            display(coeffs_df)
            if not cond_vol_df.empty:
                cond_vol_df.plot(title='Conditional volatilities (ultima janela)')
                plt.tight_layout()
                plt.show()
            if not std_resids_df.empty:
                std_resids_df.plot(title='Standardized residuals (por ativo)', subplots=True, legend=False)
                plt.tight_layout()
                plt.show()

        # compute constant conditional correlation matrix R
        if not std_resids_df.empty and std_resids_df.shape[0] > 0:
            R = std_resids_df.transpose().dot(std_resids_df) / len(std_resids_df)
            # ensure symmetric numeric matrix
            R = pd.DataFrame(R, index=tickers, columns=tickers)
            results['R'] = R
            if plot_results:
                plt.figure(figsize=(6,5))
                sns.heatmap(R, annot=True, fmt=".3f", cmap='vlag', center=0)
                plt.title('Constant Conditional Correlation matrix R (CCC)')
                plt.tight_layout()
                plt.show()
        else:
            results['R'] = None

        # compute 1-step ahead conditional covariance matrix H if requested
        H = None
        if horizon_1step and 'fitted_models' in results:
            try:
                diag = []
                for fit in fitted_models:
                    if fit is None:
                        diag.append(0.0)
                    else:
                        # forecast variance 1-step
                        vf = fit.forecast(horizon=1).variance
                        # vf may be DataFrame with last row; take last available
                        try:
                            last_var = float(vf.values[-1][0])
                        except Exception:
                            last_var = float(vf.iloc[-1,0])
                        diag.append(np.sqrt(max(last_var, 0.0)))
                D = np.zeros((len(tickers), len(tickers)))
                np.fill_diagonal(D, diag)
                if results.get('R') is not None:
                    H = D @ results['R'].values @ D
                    H = pd.DataFrame(H, index=tickers, columns=tickers)
                    results['H_1step'] = H
                    if plot_results:
                        plt.figure(figsize=(6,5))
                        sns.heatmap(H, annot=True, fmt=".3f", cmap='magma')
                        plt.title('1-step ahead conditional covariance matrix H (CCC approximation)')
                        plt.tight_layout()
                        plt.show()
                else:
                    results['H_1step'] = None
            except Exception as e:
                results['H_1step_error'] = str(e)
                print("Erro ao calcular H_1step:", e)

    # ---------------------------
    # 4) Optional: DCC-GARCH via R (rmgarch) using rpy2
    #    This block attempts to call R. If rpy2 or rmgarch not available it fails gracefully.
    # ---------------------------
    if use_dcc_r:
        try:
            # import rpy2 and activate pandas conversion
            from rpy2 import robjects
            from rpy2.robjects import pandas2ri
            pandas2ri.activate()
            r = robjects.r
            # ensure returns dataframe in R format (matrix of raw returns, not percent)
            returns_for_r = returns.copy()  # this is decimal returns (not scaled by 100)
            # If returns_for_r was percent (earlier we used returns_pct), convert back to decimal
            # We detect by scale: if mean absolute > 1 -> divide by 100
            if returns_for_r.abs().mean().max() > 1.0:
                returns_for_r = returns_for_r / 100.0
            # convert to R dataframe
            r_df = pandas2ri.py2rpy(returns_for_r.dropna())
            robjects.globalenv['returns'] = r_df
            # check if rmgarch installed
            r('if(!require("rmgarch")){install.packages("rmgarch", repos="http://cran.us.r-project.org")}')
            r('library(rmgarch)')
            # build univariate spec and dccspec
            r('library(rugarch)')
            # create univariate spec and dccspec dynamically
            n = returns_for_r.shape[1]
            # create R string to build univariate spec and multispec
            r_code = f'''
            uspec = ugarchspec(mean.model=list(armaOrder=c(0,0)), variance.model=list(garchOrder=c(1,1), model="sGARCH"), distribution.model="norm")
            multispec = multispec(replicate({n}, uspec))
            dccspec = dccspec(uspec=multispec, dccOrder=c(1,1), distribution="mvnorm")
            dccfit = dccfit(dccspec, data=returns)
            dccf = dccforecast(dccfit, n.ahead={int(dcc_n_ahead)})
            '''
            r(r_code)
            # retrieve forecasts H and R
            # H forecast is at dccf@mforecast$H
            H_r = r('dccf@mforecast$H')
            R_r = r('dccf@mforecast$R')
            # convert to pandas (best-effort: these are nested lists; convert to string to indicate success)
            results['dcc_r_forecast'] = {'H': 'available in R object dccf@mforecast$H', 'R': 'available in R object dccf@mforecast$R'}
            print("DCC-GARCH via R executado (objetos H e R disponíveis no ambiente R).")
        except Exception as e:
            results['dcc_r_error'] = str(e)
            print("Falha ao executar DCC-GARCH via R (rpy2/rmgarch). Mensagem:", e)

    # ---------------------------
    # Final
    # ---------------------------
    print("\nPipeline concluído. Result dict keys:", list(results.keys()))
    return results

# ===========================================================
# Asset Allocation: função única para 1/n, Monte Carlo e otimização
# Requisitos (Colab):
#   !pip install yfinance scipy seaborn
# Opcional (para tear sheet):
#   !pip install pyfolio
# ===========================================================
def run_asset_allocation_analysis(
    risky_assets=['FB','TSLA','TWTR','MSFT'],
    start='2018-01-01',
    end='2018-12-31',
    n_portfolios=100000,       # number of random portfolios for Monte Carlo
    trading_days=252,
    seed=42,
    do_pyfolio=False,          # try to produce simple tear sheet if pyfolio installed
    random_state=None
):
    """
    Executa:
      - download dos preços
      - cálculo de retornos diários e métricas anualizadas
      - avaliação do portfólio 1/n
      - Monte Carlo para buscar Efficient Frontier
      - otimização (SLSQP) para fronteira eficiente
      - plots: preços, retornos, nuvem de portfólios + frontier, ativos no gráfico

    Retorna dict com:
      prices, returns, avg_returns, cov_mat, montecarlo_df, ef_optimized, min_vol_portf, max_sharpe_portf, weights_sample
    """
    import numpy as np
    import pandas as pd
    import yfinance as yf
    import matplotlib.pyplot as plt
    import seaborn as sns
    import scipy.optimize as sco
    import warnings
    from datetime import datetime

    warnings.filterwarnings('ignore')
    if random_state is None:
        random_state = seed
    np.random.seed(random_state)
    sns.set_palette('cubehelix')

    # ---------------------------
    # 1) Download prices
    # ---------------------------
    tickers = list(risky_assets)
    data = yf.download(tickers, start=start, end=end, progress=False)
    if 'Adj Close' in data:
        prices = data['Adj Close'].copy()
    else:
        # fallback if single ticker format
        prices = data.copy()
        if 'Adj Close' in prices.columns:
            prices = prices['Adj Close'] if isinstance(prices['Adj Close'], pd.Series) else prices['Adj Close']

    # ensure DataFrame and drop rows with all NaN
    prices = prices.dropna(how='all')
    if isinstance(prices, pd.Series):
        prices = prices.to_frame(name=tickers[0])

    # show basic info
    print(f"Downloaded {prices.shape[0]} rows x {prices.shape[1]} assets. Period: {prices.index.min().date()} - {prices.index.max().date()}")
    results = {}
    results['prices'] = prices

    # plot prices
    plt.figure(figsize=(10,4))
    prices.plot(title='Asset prices')
    plt.tight_layout()
    plt.show()

    # ---------------------------
    # 2) Returns and stats
    # ---------------------------
    returns = prices.pct_change().dropna()
    results['returns'] = returns

    avg_returns = returns.mean() * trading_days
    cov_mat = returns.cov() * trading_days
    results['avg_returns'] = avg_returns
    results['cov_mat'] = cov_mat

    # plot returns
    plt.figure(figsize=(10,4))
    returns.plot(title='Daily returns of considered assets')
    plt.tight_layout()
    plt.show()

    # ---------------------------
    # 3) Evaluate basic 1/n portfolio
    # ---------------------------
    n_assets = len(tickers)
    weights_1n = np.repeat(1.0/n_assets, n_assets)
    portf_returns_1n = returns.dot(weights_1n)
    portf_ann_return = portf_returns_1n.mean() * trading_days
    portf_ann_vol = portf_returns_1n.std() * np.sqrt(trading_days)
    sharpe_1n = (portf_ann_return / portf_ann_vol) if portf_ann_vol != 0 else np.nan

    print("\n1/n portfolio performance:")
    print(f"  Annualized Return: {100*portf_ann_return:.2f}%")
    print(f"  Annualized Volatility: {100*portf_ann_vol:.2f}%")
    print(f"  Sharpe Ratio (no risk-free): {sharpe_1n:.4f}")

    results['1n'] = {'weights': weights_1n, 'ann_return': portf_ann_return, 'ann_vol': portf_ann_vol, 'sharpe': sharpe_1n}

    # optional pyfolio tear sheet
    if do_pyfolio:
        try:
            import pyfolio as pf
            print("\nGenerating pyfolio simple tear sheet (may open interactive plots)...")
            pf.create_simple_tear_sheet(portf_returns_1n)
        except Exception as e:
            print("pyfolio not available or error:", e)

    # ---------------------------
    # 4) Monte Carlo simulation for Efficient Frontier
    # ---------------------------
    print(f"Running Monte Carlo with {n_portfolios:,} random portfolios... (this may take a while)")
    np.random.seed(random_state)
    weights = np.random.random((n_portfolios, n_assets))
    weights /= weights.sum(axis=1)[:, np.newaxis]  # normalize

    # portfolio returns (annualized expected) and vol
    portf_rtns = weights.dot(avg_returns.values)  # shape (n_portfolios,)
    # compute vol for each portfolio
    # vectorized approach:
    vol_array = np.sqrt(np.einsum('ij,jk,ik->i', weights, cov_mat.values, weights))
    sharpe_array = portf_rtns / vol_array

    monte_df = pd.DataFrame({
        'returns': portf_rtns,
        'volatility': vol_array,
        'sharpe': sharpe_array
    })
    # keep weights sample for inspection
    weights_df = pd.DataFrame(weights, columns=tickers)

    results['montecarlo'] = monte_df
    results['weights_sample'] = weights_df

    # plot MC cloud
    fig, ax = plt.subplots(figsize=(10,6))
    sc = ax.scatter(monte_df['volatility'], monte_df['returns'], c=monte_df['sharpe'], cmap='RdYlGn', edgecolors='k', alpha=0.6)
    plt.colorbar(sc, label='Sharpe Ratio')
    ax.set_xlabel('Volatility')
    ax.set_ylabel('Expected Return')
    ax.set_title('Monte Carlo Portfolios')
    # plot individual assets
    asset_vol = np.sqrt(np.diag(cov_mat))
    asset_rtn = avg_returns.values
    markers = ['o','X','d','*','s','P','^','v']  # up to 8 different markers
    for i, t in enumerate(tickers):
        ax.scatter(asset_vol[i], asset_rtn[i], marker=markers[i % len(markers)], s=140, c='black')
        ax.text(asset_vol[i]*1.01, asset_rtn[i]*1.01, t, fontsize=9)
    plt.tight_layout()
    plt.show()

    # ---------------------------
    # 5) Extract Efficient Frontier points from Monte Carlo
    # ---------------------------
    N_POINTS = 100
    rtns_linspace = np.linspace(monte_df['returns'].min(), monte_df['returns'].max(), N_POINTS)
    ef_vol = []
    ef_returns = []
    for r_target in rtns_linspace:
        # find portfolios with approx same return (allow small tolerance)
        mask = np.isclose(np.round(monte_df['returns'], 6), np.round(r_target, 6))
        # if none, find closest
        if mask.sum() == 0:
            # find index of portfolio with closest return
            idx = (np.abs(monte_df['returns'] - r_target)).argmin()
            ef_vol.append(monte_df['volatility'].iloc[idx])
            ef_returns.append(monte_df['returns'].iloc[idx])
        else:
            ef_vol.append(monte_df.loc[mask, 'volatility'].min())
            ef_returns.append(r_target)

    ef_df_mc = pd.DataFrame({'returns': ef_returns, 'volatility': ef_vol})

    results['ef_mc'] = ef_df_mc

    # ---------------------------
    # 6) Efficient Frontier via optimization (SLSQP)
    # ---------------------------
    print("\nComputing Efficient Frontier by optimization (SLSQP)...")
    def portf_return(w, avg_rtns):
        return np.sum(avg_rtns * w)

    def portf_vol(w, avg_rtns, covm):
        return np.sqrt(np.dot(w.T, np.dot(covm, w)))

    n = n_assets
    bounds = tuple((0.0,1.0) for _ in range(n))
    cons_sum = {'type': 'eq', 'fun': lambda x: np.sum(x) - 1.0}

    # helper to minimize volatility for a target return
    def min_vol_for_return(target_return, avg_rtns, covm):
        # constraints: target return and sum(weights)=1
        cons = (
            {'type': 'eq', 'fun': lambda x: portf_return(x, avg_rtns) - target_return},
            cons_sum
        )
        x0 = np.repeat(1.0/n, n)
        res = sco.minimize(portf_vol, x0, args=(avg_rtns, covm), method='SLSQP', bounds=bounds, constraints=cons)
        return res

    rtns_range = np.linspace(monte_df['returns'].min(), monte_df['returns'].max(), 100)
    efficient_portfolios = []
    for target in rtns_range:
        try:
            r = min_vol_for_return(target, avg_returns.values, cov_mat.values)
            if r.success:
                efficient_portfolios.append(r)
            else:
                efficient_portfolios.append(None)
        except Exception:
            efficient_portfolios.append(None)

    vols_range = [res.fun if (res is not None and res.success) else np.nan for res in efficient_portfolios]
    results['efficient_portfolios_opt'] = efficient_portfolios
    # prepare optimized frontier dataframe
    ef_opt_df = pd.DataFrame({'returns': rtns_range, 'volatility': vols_range})
    results['ef_optimized'] = ef_opt_df

    # ---------------------------
    # 7) Identify min-vol and max-sharpe portfolios (on Monte Carlo + optimization)
    # ---------------------------
    # From MC:
    idx_min_vol = monte_df['volatility'].idxmin()
    min_vol_mc = {'volatility': monte_df.loc[idx_min_vol,'volatility'],
                  'returns': monte_df.loc[idx_min_vol,'returns'],
                  'weights': weights_df.loc[idx_min_vol].values}

    idx_max_sh = monte_df['sharpe'].idxmax()
    max_sh_mc = {'volatility': monte_df.loc[idx_max_sh,'volatility'],
                 'returns': monte_df.loc[idx_max_sh,'returns'],
                 'weights': weights_df.loc[idx_max_sh].values}

    results['min_vol_mc'] = min_vol_mc
    results['max_sharpe_mc'] = max_sh_mc

    # From optimization: find the feasible best sharpe across ef_opt (compute weights for each efficient portfolio)
    opt_weights = []
    for res in efficient_portfolios:
        if res is None or (not hasattr(res,'x')):
            opt_weights.append(None)
        else:
            opt_weights.append(res.x)
    # compute sharpe for each (avoid divide by zero)
    opt_sharpes = []
    for w in opt_weights:
        if w is None:
            opt_sharpes.append(np.nan)
        else:
            rtn = portf_return(w, avg_returns.values)
            vol = portf_vol(w, avg_returns.values, cov_mat.values)
            opt_sharpes.append(rtn/vol if vol>0 else np.nan)
    # best optimized sharpe
    if len(opt_sharpes)>0 and not all(np.isnan(opt_sharpes)):
        best_idx = int(np.nanargmax(opt_sharpes))
        best_res = efficient_portfolios[best_idx]
        max_sharp_opt = {'returns': rtns_range[best_idx],
                         'volatility': vols_range[best_idx],
                         'weights': best_res.x if (best_res is not None and best_res.success) else None,
                         'sharpe': opt_sharpes[best_idx]}
    else:
        max_sharp_opt = None

    results['max_sharpe_opt'] = max_sharp_opt

    # ---------------------------
    # 8) Plot Efficient Frontier (MC and Optimized)
    # ---------------------------
    fig, ax = plt.subplots(figsize=(10,6))
    sc = ax.scatter(monte_df['volatility'], monte_df['returns'], c=monte_df['sharpe'], cmap='RdYlGn', s=8, alpha=0.5, edgecolors='k')
    ax.plot(ef_df_mc['volatility'], ef_df_mc['returns'], 'b--', lw=1.5, label='EF (MC approx)')
    ax.plot(ef_opt_df['volatility'], ef_opt_df['returns'], 'r-', lw=2, label='EF (optimized)')
    # highlight min-vol and max-sharpe (MC)
    ax.scatter(min_vol_mc['volatility'], min_vol_mc['returns'], marker='*', s=200, c='cyan', edgecolor='k', label='Min Vol (MC)')
    ax.scatter(max_sh_mc['volatility'], max_sh_mc['returns'], marker='*', s=200, c='gold', edgecolor='k', label='Max Sharpe (MC)')
    if max_sharp_opt is not None:
        ax.scatter(max_sharp_opt['volatility'], max_sharp_opt['returns'], marker='D', s=140, c='magenta', edgecolor='k', label='Max Sharpe (opt)')
    # plot assets
    for i, t in enumerate(tickers):
        ax.scatter(asset_vol[i], asset_rtn[i], marker=markers[i % len(markers)], s=140, c='black')
        ax.text(asset_vol[i]*1.01, asset_rtn[i]*1.01, t, fontsize=9)
    ax.set_xlabel('Volatility')
    ax.set_ylabel('Expected Return')
    ax.set_title('Efficient Frontier: Monte Carlo vs Optimized')
    ax.legend()
    plt.tight_layout()
    plt.show()

    # ---------------------------
    # 9) Print summary of optimized min-vol portfolio (if available)
    # ---------------------------
    # find min-vol from optimized results
    try:
        idx_min_opt = np.nanargmin(vols_range)
        min_opt_res = efficient_portfolios[idx_min_opt]
        if min_opt_res is not None and min_opt_res.success:
            w_min_opt = min_opt_res.x
            rtn_min_opt = rtns_range[idx_min_opt]
            vol_min_opt = vols_range[idx_min_opt]
            print("\nOptimized min-vol portfolio (SLSQP):")
            print(f"  Return: {100*rtn_min_opt:.2f}%  Volatility: {100*vol_min_opt:.2f}%")
            print("  Weights:")
            for t,w in zip(tickers, w_min_opt):
                print(f"    {t}: {100*w:.2f}%")
            results['min_vol_opt'] = {'weights': w_min_opt, 'returns': rtn_min_opt, 'volatility': vol_min_opt}
        else:
            results['min_vol_opt'] = None
    except Exception as e:
        results['min_vol_opt'] = None

    # final results
    results['montecarlo_df'] = monte_df
    results['weights_sample_df'] = weights_df
    results['ef_mc_df'] = ef_df_mc
    results['ef_opt_df'] = ef_opt_df
    results['tickers'] = tickers

    print("\nDone — returning results dictionary.")
    return results

# MACHINE LEARNING PIPELINE — CREDIT CARD DEFAULT DETECTION
# ============================================================

def credit_card_default_pipeline(csv_path: str):
    """
    Executa todo o pipeline de Machine Learning para o dataset
    'credit_card_default.csv', incluindo:
      - carregamento e otimização de memória
      - análise exploratória básica (EDA)
      - imputação de valores ausentes
      - encoding de variáveis categóricas
      - criação de pipeline sklearn
      - ajuste do modelo DecisionTree
      - validação cruzada
      - GridSearch e RandomizedSearch
      - relatório final de métricas

    Parâmetros
    ----------
    csv_path : str
        Caminho do arquivo CSV (ex: 'credit_card_default.csv')
    """
    import pandas as pd
    import numpy as np
    import seaborn as sns
    import matplotlib.pyplot as plt
    from sklearn.model_selection import train_test_split, GridSearchCV, RandomizedSearchCV, cross_validate, StratifiedKFold
    from sklearn.tree import DecisionTreeClassifier
    from sklearn.impute import SimpleImputer
    from sklearn.preprocessing import OneHotEncoder
    from sklearn.compose import ColumnTransformer
    from sklearn.pipeline import Pipeline
    from sklearn import metrics
    import warnings
    warnings.filterwarnings('ignore')

    plt.style.use('seaborn-v0_8') # Updated style name
    plt.rcParams['figure.figsize'] = [8, 4.5]
    plt.rcParams['figure.dpi'] = 300

    # ============================================================
    # 1. LOAD DATA
    # ============================================================
    df = pd.read_csv(csv_path, index_col=0, na_values='')
    print(f"📂 Dataset carregado: {len(df)} linhas, {df.shape[1]} colunas")

    X = df.copy()
    y = X.pop('default_payment_next_month')

    # ============================================================
    # 2. SPLIT TRAIN / TEST
    # ============================================================
    X_train, X_test, y_train, y_test = train_test_split(
        X, y, test_size=0.2, stratify=y, random_state=42
    )

    # ============================================================
    # 3. FEATURES
    # ============================================================
    num_features = X_train.select_dtypes(include='number').columns.tolist()
    cat_features = X_train.select_dtypes(include='object').columns.tolist()

    # ============================================================
    # 4. PIPELINES DE PRÉ-PROCESSAMENTO
    # ============================================================
    num_pipeline = Pipeline(steps=[
        ('imputer', SimpleImputer(strategy='median'))
    ])

    cat_pipeline = Pipeline(steps=[
        ('imputer', SimpleImputer(strategy='most_frequent')),
        ('onehot', OneHotEncoder(handle_unknown='ignore', sparse=False, drop='first'))
    ])

    preprocessor = ColumnTransformer(transformers=[
        ('num', num_pipeline, num_features),
        ('cat', cat_pipeline, cat_features)
    ])

    # ============================================================
    # 5. CLASSIFICADOR
    # ============================================================
    model = DecisionTreeClassifier(random_state=42)

    pipeline = Pipeline(steps=[
        ('preprocessor', preprocessor),
        ('classifier', model)
    ])

    # ============================================================
    # 6. TREINAMENTO BÁSICO
    # ============================================================
    pipeline.fit(X_train, y_train)
    y_pred = pipeline.predict(X_test)

    acc = metrics.accuracy_score(y_test, y_pred)
    rec = metrics.recall_score(y_test, y_pred)
    prec = metrics.precision_score(y_test, y_pred)
    auc = metrics.roc_auc_score(y_test, y_pred)

    print("\n📊 Desempenho inicial:")
    print(f"  - Acurácia: {acc:.4f}")
    print(f"  - Precisão: {prec:.4f}")
    print(f"  - Recall:   {rec:.4f}")
    print(f"  - AUC:      {auc:.4f}")

    # ============================================================
    # 7. CROSS VALIDATION
    # ============================================================
    k_fold = StratifiedKFold(5, shuffle=True, random_state=42)
    cv_results = cross_validate(pipeline, X_train, y_train, cv=k_fold,
                                scoring=['accuracy', 'precision', 'recall', 'roc_auc'])
    print("\n📈 Cross-Validation Média:")
    for key, val in cv_results.items():
        if 'test' in key:
            print(f"  - {key.replace('test_', '').capitalize()}: {np.mean(val):.4f}")

    # ============================================================
    # 8. GRID SEARCH
    # ============================================================
    param_grid = {
        'classifier__criterion': ['entropy', 'gini'],
        'classifier__max_depth': range(3, 11),
        'classifier__min_samples_leaf': range(2, 11)
    }

    grid_search = GridSearchCV(pipeline, param_grid, scoring='recall',
                               cv=k_fold, n_jobs=-1, verbose=1)
    grid_search.fit(X_train, y_train)

    print("\n🔍 Melhor modelo (Grid Search):")
    print(f"  - Parâmetros: {grid_search.best_params_}")
    print(f"  - Recall (Treino): {grid_search.best_score_:.4f}")
    print(f"  - Recall (Teste):  {metrics.recall_score(y_test, grid_search.predict(X_test)):.4f}")

    # ============================================================
    # 9. RANDOMIZED SEARCH
    # ============================================================
    rand_search = RandomizedSearchCV(pipeline, param_grid, scoring='recall',
                                     cv=k_fold, n_jobs=-1, n_iter=50, random_state=42, verbose=1)
    rand_search.fit(X_train, y_train)

    print("\n🎲 Melhor modelo (Randomized Search):")
    print(f"  - Parâmetros: {rand_search.best_params_}")
    print(f"  - Recall (Treino): {rand_search.best_score_:.4f}")
    print(f"  - Recall (Teste):  {metrics.recall_score(y_test, rand_search.predict(X_test)):.4f}")

    # ============================================================
    # 10. GRÁFICO DE MATRIZ DE CONFUSÃO
    # ============================================================
    conf = metrics.confusion_matrix(y_test, rand_search.predict(X_test))
    sns.heatmap(conf, annot=True, fmt='d', cmap='Blues')
    plt.title('Matriz de Confusão (Randomized Search)')
    plt.xlabel('Predito')
    plt.ylabel('Real')
    plt.show()

    print("\n✅ Pipeline completo executado com sucesso.")

# ============================================================
# MACHINE LEARNING PIPELINE — CREDIT CARD DEFAULT DETECTION
# ============================================================

def credit_card_default_pipeline(csv_path: str):
    """
    Executa todo o pipeline de Machine Learning para o dataset
    'credit_card_default.csv', incluindo:
      - carregamento e otimização de memória
      - análise exploratória básica (EDA)
      - imputação de valores ausentes
      - encoding de variáveis categóricas
      - criação de pipeline sklearn
      - ajuste do modelo DecisionTree
      - validação cruzada
      - GridSearch e RandomizedSearch
      - relatório final de métricas

    Parâmetros
    ----------
    csv_path : str
        Caminho do arquivo CSV (ex: 'credit_card_default.csv')
    """
    import pandas as pd
    import numpy as np
    import seaborn as sns
    import matplotlib.pyplot as plt
    from sklearn.model_selection import train_test_split, GridSearchCV, RandomizedSearchCV, cross_validate, StratifiedKFold
    from sklearn.tree import DecisionTreeClassifier
    from sklearn.impute import SimpleImputer
    from sklearn.preprocessing import OneHotEncoder
    from sklearn.compose import ColumnTransformer
    from sklearn.pipeline import Pipeline
    from sklearn import metrics
    import warnings
    warnings.filterwarnings('ignore')

    plt.style.use('seaborn')
    plt.rcParams['figure.figsize'] = [8, 4.5]
    plt.rcParams['figure.dpi'] = 300

    # ============================================================
    # 1. LOAD DATA
    # ============================================================
    df = pd.read_csv(csv_path, index_col=0, na_values='')
    print(f"📂 Dataset carregado: {len(df)} linhas, {df.shape[1]} colunas")

    X = df.copy()
    y = X.pop('default_payment_next_month')

    # ============================================================
    # 2. SPLIT TRAIN / TEST
    # ============================================================
    X_train, X_test, y_train, y_test = train_test_split(
        X, y, test_size=0.2, stratify=y, random_state=42
    )

    # ============================================================
    # 3. FEATURES
    # ============================================================
    num_features = X_train.select_dtypes(include='number').columns.tolist()
    cat_features = X_train.select_dtypes(include='object').columns.tolist()

    # ============================================================
    # 4. PIPELINES DE PRÉ-PROCESSAMENTO
    # ============================================================
    num_pipeline = Pipeline(steps=[
        ('imputer', SimpleImputer(strategy='median'))
    ])

    cat_pipeline = Pipeline(steps=[
        ('imputer', SimpleImputer(strategy='most_frequent')),
        ('onehot', OneHotEncoder(handle_unknown='ignore', sparse=False, drop='first'))
    ])

    preprocessor = ColumnTransformer(transformers=[
        ('num', num_pipeline, num_features),
        ('cat', cat_pipeline, cat_features)
    ])

    # ============================================================
    # 5. CLASSIFICADOR
    # ============================================================
    model = DecisionTreeClassifier(random_state=42)

    pipeline = Pipeline(steps=[
        ('preprocessor', preprocessor),
        ('classifier', model)
    ])

    # ============================================================
    # 6. TREINAMENTO BÁSICO
    # ============================================================
    pipeline.fit(X_train, y_train)
    y_pred = pipeline.predict(X_test)

    acc = metrics.accuracy_score(y_test, y_pred)
    rec = metrics.recall_score(y_test, y_pred)
    prec = metrics.precision_score(y_test, y_pred)
    auc = metrics.roc_auc_score(y_test, y_pred)

    print("\n📊 Desempenho inicial:")
    print(f"  - Acurácia: {acc:.4f}")
    print(f"  - Precisão: {prec:.4f}")
    print(f"  - Recall:   {rec:.4f}")
    print(f"  - AUC:      {auc:.4f}")

    # ============================================================
    # 7. CROSS VALIDATION
    # ============================================================
    k_fold = StratifiedKFold(5, shuffle=True, random_state=42)
    cv_results = cross_validate(pipeline, X_train, y_train, cv=k_fold,
                                scoring=['accuracy', 'precision', 'recall', 'roc_auc'])
    print("\n📈 Cross-Validation Média:")
    for key, val in cv_results.items():
        if 'test' in key:
            print(f"  - {key.replace('test_', '').capitalize()}: {np.mean(val):.4f}")

    # ============================================================
    # 8. GRID SEARCH
    # ============================================================
    param_grid = {
        'classifier__criterion': ['entropy', 'gini'],
        'classifier__max_depth': range(3, 11),
        'classifier__min_samples_leaf': range(2, 11)
    }

    grid_search = GridSearchCV(pipeline, param_grid, scoring='recall',
                               cv=k_fold, n_jobs=-1, verbose=1)
    grid_search.fit(X_train, y_train)

    print("\n🔍 Melhor modelo (Grid Search):")
    print(f"  - Parâmetros: {grid_search.best_params_}")
    print(f"  - Recall (Treino): {grid_search.best_score_:.4f}")
    print(f"  - Recall (Teste):  {metrics.recall_score(y_test, grid_search.predict(X_test)):.4f}")

    # ============================================================
    # 9. RANDOMIZED SEARCH
    # ============================================================
    rand_search = RandomizedSearchCV(pipeline, param_grid, scoring='recall',
                                     cv=k_fold, n_jobs=-1, n_iter=50, random_state=42, verbose=1)
    rand_search.fit(X_train, y_train)

    print("\n🎲 Melhor modelo (Randomized Search):")
    print(f"  - Parâmetros: {rand_search.best_params_}")
    print(f"  - Recall (Treino): {rand_search.best_score_:.4f}")
    print(f"  - Recall (Teste):  {metrics.recall_score(y_test, rand_search.predict(X_test)):.4f}")

    # ============================================================
    # 10. GRÁFICO DE MATRIZ DE CONFUSÃO
    # ============================================================
    conf = metrics.confusion_matrix(y_test, rand_search.predict(X_test))
    sns.heatmap(conf, annot=True, fmt='d', cmap='Blues')
    plt.title('Matriz de Confusão (Randomized Search)')
    plt.xlabel('Predito')
    plt.ylabel('Real')
    plt.show()

    print("\n✅ Pipeline completo executado com sucesso.")
