# Portfólio em Ciência de Dados - Lucas Borges

Um portfólio interativo showcasing projetos e análises em Ciência de Dados, desenvolvido usando Quarto e otimizado para hospedagem no Hugging Face Spaces.

## 🚀 Demonstração

Este projeto está hospedado no Hugging Face Spaces: [https://huggingface.co/spaces/lucasborges/portfolio-data-science](https://huggingface.co/spaces/lucasborges/portfolio-data-science)

## 📋 Sobre

Este portfólio apresenta projetos desenvolvidos durante minha jornada em Ciência de Dados, incluindo:

- **Análise Exploratória de Dados (EDA)** de competições Kaggle
- **Implementação de modelos** de Machine Learning
- **Visualizações interativas** com Plotly e Seaborn
- **Documentação técnica** completa e bem estruturada

## 🛠️ Tecnologias Utilizadas

- **Quarto** - Para geração de documentos dinâmicos
- **Python** - Linguagem principal para análise de dados
- **Pandas & NumPy** - Manipulação de dados
- **Matplotlib & Seaborn** - Visualizações estáticas
- **Plotly** - Visualizações interativas
- **Scikit-learn** - Machine Learning
- **Statsmodels** - Análise estatística

## 🚀 Como Hospedar no Hugging Face Spaces

Para hospedar este projeto no Hugging Face Spaces:

1. **Crie um novo Space:**
   - Vá para [huggingface.co/new-space](https://huggingface.co/new-space)
   - Escolha "Static HTML" como o template
   - Dê um nome para seu space (ex: `seu-usuario/portfolio-data-science`)

2. **Faça o upload dos arquivos:**
   - Faça o upload de todos os arquivos do seu projeto
   - Inclua o `requirements.txt` caso precise de dependências

3. **Configure o build (opcional):**
   - Se precisar de Quarto, adicione um `Dockerfile`
   - Caso contrário, apenas os arquivos estáticos são suficientes

4. **Configure o README do Space:**
   - Adicione uma descrição clara do seu projeto
   - Inclua links para demonstração e documentação
